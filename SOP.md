# 钢铁雄心4玩家指南 - 操作SOP文档

## 1. 网站概述

本网站是《钢铁雄心4》玩家的一站式资源、攻略与社区中心，提供以下核心功能：

- 首页导航与游戏概览
- 资源中心（游戏本体、DLC、MOD资源）
- 攻略索引（分类浏览、添加、编辑攻略）
- 社区交流平台推荐

## 2. 用户浏览指南

### 2.1 页面导航

1. **顶部导航栏**
   - 点击左侧Logo返回首页
   - 使用中间导航链接在"首页"、"资源"、"攻略"、"社区"页面间切换
   - 右侧可切换暗黑/明亮主题模式

2. **移动端导航**
   - 点击右上角菜单按钮展开导航菜单
   - 选择目标页面后，菜单自动收起

### 2.2 主题切换

1. 点击顶部导航栏右侧的月亮/太阳图标
2. 网站将在暗黑模式与明亮模式间切换
3. 主题设置会自动保存到本地存储

## 3. 攻略管理操作流程

### 3.1 浏览攻略

1. 进入攻略页面（点击顶部导航栏"攻略"链接）
2. 使用顶部分类筛选器选择特定类型的攻略
3. 滚动浏览攻略列表
4. 每个攻略卡片显示标题、难度等级、描述、作者和阅读时长

### 3.2 添加新攻略

1. 在攻略页面顶部找到"快速生成攻略"表单
2. 填写必填字段（带红色星号）：
   - 攻略标题
   - 攻略内容
3. 选择可选字段：
   - 难度等级（入门/中级/高级）
   - 预计阅读时长（1-10分钟）
   - 上传相关图片（可选）
4. 点击"添加攻略"按钮
5. 新攻略将显示在攻略列表顶部
6. 系统会显示"攻略添加成功"提示

### 3.3 编辑现有攻略

1. 在攻略列表中，找到需要编辑的攻略卡片
2. 点击攻略卡片右下角的编辑图标（铅笔形状）
3. 表单将填充该攻略的现有内容
4. 修改所需字段
5. 点击"保存攻略"按钮应用更改
   - 或点击"取消编辑"按钮放弃更改
6. 系统会显示"攻略更新成功"提示

## 4. 资源访问指南

### 4.1 游戏本体

1. 进入资源页面（点击顶部导航栏"资源"链接）
2. 选择"游戏本体"标签
3. 浏览可用的游戏版本选项
4. 点击相应的"下载"或"购买"按钮获取游戏

### 4.2 DLC内容

1. 进入资源页面
2. 选择"DLC内容"标签
3. 浏览可用的DLC列表（按发布时间排序）
4. 查看每个DLC的详细介绍和文件大小

### 4.3 MOD模组

1. 进入资源页面
2. 选择"MOD模组"标签
3. 浏览推荐的MOD列表
4. 点击页面顶部的"前往Steam创意工坊"按钮访问更多MOD
5. 在Steam创意工坊中订阅和管理MOD

## 5. 社区平台访问

1. 进入社区页面（点击顶部导航栏"社区"链接）
2. 浏览推荐的社区平台列表
3. 每个社区卡片显示名称、简介、成员数量和平台类型

## 6. 开发与维护指南

### 6.1 代码保存与版本控制

1. **本地备份**
   - 复制整个项目文件夹到安全位置

2. **Git版本控制（推荐）**
   ```bash
   # 初始化Git仓库（如未初始化）
   git init
   
   # 添加所有文件
   git add .
   
   # 提交更改
   git commit -m "保存当前版本"
   
   # 推送到远程仓库（如配置了远程仓库）
   git push origin master
   ```

### 6.2 修改代码

1. 使用代码编辑器（如VS Code）打开项目文件夹
2. 根据需要修改相应文件：
   - 页面文件：`src/pages/` 目录下
   - 组件文件：`src/components/` 目录下
   - 样式：通过Tailwind CSS类进行样式调整
3. 修改完成后，运行开发服务器验证更改

### 6.3 本地开发

1. 确保已安装Node.js和pnpm
2. 在项目根目录运行以下命令：
   ```bash
   # 安装依赖
   pnpm install
   
   # 启动开发服务器
   pnpm dev
   ```
3. 打开浏览器访问 `http://localhost:3000` 查看效果

### 6.4 构建与部署

1. **构建项目**
   ```bash
   # 在项目根目录运行
   pnpm build
   ```
   - 构建产物将生成在 `dist` 目录中

2. **部署到静态托管服务**
   - 将 `dist` 目录中的所有文件上传到您选择的托管服务
   - 支持的托管服务包括：Vercel、Netlify、GitHub Pages、阿里云OSS等

## 7. 常见问题解决

### 7.1 依赖安装问题

- **问题**：运行 `pnpm install` 失败
- **解决方法**：
  ```bash
  # 删除现有依赖和锁文件
  rm -rf node_modules pnpm-lock.yaml
  
  # 重新安装依赖
  pnpm install
  ```

### 7.2 构建失败

- **问题**：运行 `pnpm build` 失败
- **解决方法**：
  1. 检查控制台错误信息，定位具体问题
  2. 确保代码没有语法错误或类型错误
  3. 尝试删除 `node_modules` 和 `pnpm-lock.yaml` 后重新安装依赖

### 7.3 页面显示问题

- **问题**：页面布局错乱或元素不显示
- **解决方法**：
  1. 清除浏览器缓存后刷新页面
  2. 检查是否有CSS冲突
  3. 验证响应式设计在不同设备上的表现

## 8. 快捷键

目前网站不支持自定义快捷键，所有操作均通过鼠标或触摸屏完成。
